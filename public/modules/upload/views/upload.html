Creating Your Own CSV File

Our CSV holds only the basic required fields, if you want to add your filters and fields just add "Filter_" or "Fields_" to your filter/field name and add the new keyword to the template CSV at the 1st line after a comma.

Importing Your Own CSV File

To import your file the file format must be csv format only.The following steps guide you how to save file with csv extention.

1. Save your spread sheet as a .CSV. Select the "Save As" option and then select CSV (Comma delimited).

2. Make sure there is a .csv extension after the file name and then click save.

Now your file is all ready to import! Browse the file in your respective folder where the csv file is saved.
